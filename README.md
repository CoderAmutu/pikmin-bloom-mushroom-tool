# 皮克敏蘑菇重生推算器

一個以 **Pikmin Bloom 蘑菇挑戰場景** 為核心設計的純前端小工具，協助玩家快速記錄各地點蘑菇的剩餘時間，並自動推算重生時間，讓整理與追蹤多筆蘑菇資訊變得更直覺、更有效率。

本工具採用 **HTML / CSS / JavaScript** 製作，不需後端、不需安裝套件、也不需資料庫，只要直接開啟 `index.html` 即可使用。所有清單資料、提醒設定與地點標籤皆會儲存在瀏覽器本機 `localStorage` 中，重新整理頁面後仍可保留。

---

## 專案特色

這個工具的設計重點不是單純倒數，而是把玩家在實際記錄蘑菇時常遇到的幾個痛點整合起來：

- 同時追蹤多個地點的蘑菇
- 快速輸入剩餘時間並立即開始倒數
- 自動推算蘑菇重生時間（剩餘時間結束後再加 5 分鐘）
- 一眼看出「下一朵最先冒出來的蘑菇」
- 在接近重生前給予提醒，避免錯過時間
- 保留常用地點標籤，減少重複輸入
- 可排序、可複製、可清空，方便整理分享

整體介面以 **皮克敏風格的配色與背景視覺** 為主，並兼顧桌機與手機瀏覽體驗，讓工具既實用也保有遊戲感。

## 目前功能

### 1. 多筆蘑菇清單管理
- 可新增多筆蘑菇資料
- 每筆資料包含：
  - 地點名稱
  - 輸入剩餘時間（時 / 分 / 秒）
  - 即時倒數顯示
  - 自動推算的重生時間
- 可單筆刪除資料

### 2. 蘑菇重生時間自動推算
輸入剩餘時間後，系統會以「**輸入當下那一秒**」作為起點開始倒數，並在倒數結束後自動再加上 **5 分鐘**，作為蘑菇重生時間。

這樣使用者不需要自行心算或另外開計時器，即可直接知道下一次可重新挑戰的時間點。

### 3. 即時台北時間顯示
頁面會持續顯示目前台北時間，方便玩家直接對照現實時間與蘑菇推算結果。

### 4. 即將冒出來的蘑菇提示卡
系統會自動找出目前清單中「**尚未過期且最早重生**」的一筆資料，並顯示在頁面右上角的資訊卡中，包含：

- 地點名稱
- 推算重生時間
- 距離重生還剩多久

當畫面往下捲動時，也會顯示一張 **浮動提示卡**，避免使用者在瀏覽長清單時看不到最重要的下一筆資訊。

### 5. 提前提醒功能
可設定是否開啟「提前提醒」，並自訂在重生前幾秒開始提醒。

目前支援：
- 提前提醒開關
- 提醒秒數設定（1～60 秒）
- 重生前持續提醒模式
- 關閉持續提醒模式，只提醒一次

### 6. 頻繁提醒（每 5 秒）
若開啟頻繁提醒，系統會在進入提醒區間後，每 **5 秒** 提醒一次，直到蘑菇重生或使用者手動關閉該筆提醒。

這個設計適合需要高注意力追蹤、怕錯過時間點的使用情境。

### 7. 音效提醒與音量調整
工具內建提醒音效，並支援音量滑桿調整。

說明：
- 提前提醒與重生提醒會使用不同節奏的音效
- 首次使用時，瀏覽器可能需要先點擊頁面，才能啟用音效播放
- 音量設定會保存於瀏覽器中

### 8. 地點標籤功能
常去的地點可加入標籤清單，後續只要點擊標籤，即可快速填入最新新增那一列的地點欄位。

支援：
- 新增標籤
- 記住標籤
- 刪除標籤
- 重複加入相同標籤時，自動移到前面

這能大幅減少重複輸入常用地點的時間。

### 9. 排序功能
提供「依重生時間排序」按鈕，排序邏輯如下：

1. **尚未重生的蘑菇** 會排在前面，並依照最早重生時間排序
2. **已過期的蘑菇** 會沉到後面
3. **尚未輸入時間的空白列** 會排在最後

這個排序方式更符合實際使用需求，能讓真正需要優先關注的蘑菇自然排在前面。

### 10. 複製功能
支援：
- 單筆複製
- 全部複製

複製格式為：

`地點名稱|推算重生時間：HH:MM:SS`

方便快速貼到聊天、記事本或其他紀錄工具中。

### 11. 清單資料自動保存
以下資料都會自動儲存在瀏覽器本機：

- 蘑菇清單
- 地點標籤
- 提前提醒設定
- 頻繁提醒設定
- 音量設定

重新整理頁面或重新開啟瀏覽器後，先前紀錄仍可保留，不必重新輸入。

### 12. 響應式介面設計
目前畫面已針對不同螢幕尺寸做版面調整，包含：

- 桌機版資訊卡與清單排版
- 手機版區塊重排
- 浮動資訊卡輔助瀏覽
- 背景圖與主要介面分層呈現

## 使用方式

### 直接使用
1. 開啟 `index.html`
2. 在「蘑菇清單」中輸入地點名稱
3. 輸入看到的剩餘時間（時 / 分 / 秒）
4. 系統會立即開始倒數，並自動推算重生時間
5. 視需要使用排序、提醒、標籤與複製功能

### 建議使用流程
1. 先新增一筆蘑菇
2. 輸入地點
3. 輸入遊戲內看到的剩餘時間
4. 若是常用地點，可按「加入標籤」
5. 當有多筆資料時，使用「依重生時間排序」整理優先順序
6. 開啟提前提醒，避免錯過重生時間

## 技術說明

### 前端技術
- HTML5
- CSS3
- Vanilla JavaScript

### 使用的瀏覽器能力
- `localStorage`：保存清單與設定
- `Intl.DateTimeFormat`：格式化台北時間
- `Web Audio API`：播放提醒音效
- `crypto.randomUUID()`：建立每筆資料的唯一 ID
- `navigator.clipboard`：複製文字到剪貼簿

### 專案特性
- 無框架
- 無建置工具
- 無安裝依賴
- 可直接部署為靜態網站
